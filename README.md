# Sass Sample

The simple example for how to use sass.

#

# How to Run Project

## Download Codes

```
git clone https://github.com/dori-dev/sass-sample.git
```

```
cd sass-sample
```

## Install Packages

```
npm install
```

## Compile Sass

```
sass index.sass style.css
```
